# Basic File Editing with vi (LPIC-1 103.8)

## Goal

Practice editing Linux text and configuration files using `vi` or `vim`.

This module focuses on navigating files, switching between editor modes, inserting and modifying text, searching, copying, deleting, pasting, saving, and configuring the default system editor.

## Skills Practiced

- Opening files with `vi`
- Understanding Normal, Insert, and Command-line modes
- Navigating with `h`, `j`, `k`, and `l`
- Moving between words and lines
- Inserting and appending text
- Opening new lines
- Deleting characters, words, and lines
- Copying and pasting text
- Searching forward and backward
- Saving and quitting safely
- Running shell commands from inside `vi`
- Configuring `EDITOR` and `VISUAL`
- Awareness of Vim, Nano, Emacs, Micro, and Neovim
