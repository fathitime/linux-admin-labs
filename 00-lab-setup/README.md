# Linux Lab Setup

## Goal

Set up a Linux practice environment for LPIC-1 hands-on labs.

## Distributions Used

- Ubuntu
- Fedora

## Lab Purpose

This lab environment is used to practice Linux administration commands, boot process concepts, systemd targets, hardware inspection, and other LPIC-1 topics.

## Notes

- Ubuntu is useful for Debian-based package management and common server environments.
- Fedora is useful for Red Hat-based concepts and systemd practice.
- Both environments help compare Linux distributions and administration workflows.
