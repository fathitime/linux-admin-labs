# Regular Expressions and Text Search (LPIC-1 103.7)

## Goal

Practice searching, filtering, validating, and modifying text using regular expressions with `grep` and `sed`.

This module focuses on practical Linux administration use cases such as log analysis, configuration inspection, data validation, and text substitution.

## Skills Practiced

- Basic Regular Expressions (BRE)
- Extended Regular Expressions (ERE)
- Literal and fixed-string searches
- Anchors: `^` and `$`
- Character classes and ranges
- Negated character classes
- Quantifiers
- Alternation and grouping
- Recursive filesystem searches
- Case-insensitive and inverse matching
- Text substitution and deletion with `sed`
- Basic log analysis with regular expressions

## Tools Used

- `grep`
- `grep -E`
- `grep -F`
- `sed -E`
- `regex(7)`
