# Hardware Information Commands

## Goal

Practice basic Linux commands for inspecting hardware and system information.

## Commands Practiced

```bash
uname -a
uname -r
lscpu
lsblk
lsusb
lspci
free -h
df -h
cat /proc/cpuinfo
cat /proc/meminfo

```text
Notes
uname -r shows the Linux kernel version.
lscpu shows CPU architecture information.
lsblk lists block devices such as disks and partitions.
lsusb lists USB devices.
lspci lists PCI devices.
free -h shows memory usage in a human-readable format.
df -h shows disk usage.
/proc/cpuinfo contains CPU details.
/proc/meminfo contains memory details.
