# Linux as a Virtual Machine Guest

## Goal

Understand Linux installation and usage as a virtual machine guest.

## Key Concepts

- Virtual machine
- Guest operating system
- Host operating system
- Virtual disk
- Virtual CPU
- Virtual memory
- Network adapter modes
- Guest tools

## Lab Notes

I use Linux virtual machines to safely practice LPIC-1 topics, including disk management, package management, boot process, and system administration commands.

## Benefits of Using Virtual Machines

- Safe practice environment
- Easy snapshot and rollback
- No risk to the main operating system
- Ability to compare multiple Linux distributions
- Useful for DevOps and cloud learning
