# Basic File Management (LPIC-1 103.3)

## Goal

Practice basic Linux file and directory management commands as part of LPIC-1 103.3.

This module focuses on working with files, directories, wildcards, file search, archiving, compression, and safe command-line operations.

---

## Skills Practiced

- Creating files and directories
- Copying, moving, and removing files
- Recursive file and directory operations
- File globbing and wildcards
- Finding files by name, type, size, and time
- Using `find` with actions
- Creating archives with `tar` and `cpio`
- Compressing and decompressing files
- Understanding safe usage of `dd`

---

## Lab Approach

All commands are designed to be practiced inside a safe lab directory, not on real production paths.

Recommended lab path:

```bash
~/lpic1-labs/file-management-lab
```
