# Practice Summary

## LPIC-1 Topic

103.5 Create, monitor, and kill processes

## What I Practiced

- Running foreground and background jobs
- Suspending and resuming jobs
- Viewing job numbers and process IDs
- Inspecting processes with `ps`
- Sorting processes by CPU and memory usage
- Monitoring resources with `top`, `free`, and `uptime`
- Locating processes with `pgrep`
- Repeating monitoring commands with `watch`
- Sending `SIGTERM` and `SIGKILL`
- Terminating processes by name with `killall` and `pkill`
- Keeping commands active with `nohup`
- Working with persistent sessions using `screen` and `tmux`

## Real-World Use Cases

- Diagnosing high CPU or memory usage
- Managing long-running commands
- Troubleshooting stuck processes
- Running jobs after an SSH disconnect
- Monitoring applications and servers
- Managing multiple terminal sessions remotely
